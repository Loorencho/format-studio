# Сайт для студии ремонта

Лендинг студии ремонта **«Студия Формат»** с админ-панелью для заявок.

## Запуск

```bash
npm install
npm run dev
```

- **Клиентский сайт:** http://localhost:5173/
- **Админ-панель:** http://localhost:5173/admin
- **Пароль админа:** `format2025` (можно изменить через переменную `ADMIN_PASSWORD`)

Команда `npm run dev` запускает одновременно фронтенд (Vite) и бэкенд (Express).

## Как проверить заявки

1. Откройте http://localhost:5173/ и отправьте форму в разделе «Контакты»
2. Откройте http://localhost:5173/admin, войдите с паролем `format2025`
3. Заявка появится в списке

## Продакшен

```bash
npm run build
npm run start
```

Сайт и API будут доступны на http://localhost:3001

## API

| Метод | Путь | Описание |
|-------|------|----------|
| POST | `/api/leads` | Создать заявку (публичный) |
| POST | `/api/admin/login` | Вход в админку |
| GET | `/api/leads` | Список заявок (требует токен) |
| PATCH | `/api/leads/:id` | Изменить статус (`new` / `done`) |
| DELETE | `/api/leads/:id` | Удалить заявку |

## Технологии

- React 19 + React Router
- Vite 8
- Express + JSON-файл для хранения заявок
